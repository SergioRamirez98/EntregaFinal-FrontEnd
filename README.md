## PIZZERÍA EMILIO!

Este repositorio contiene información ficticia de distintos lugares de comidas, asimilados a un vecino el cual se dedica a este rubro

## Propósito del Proyecto

El objetivo principal de este sitio web es servir como portal de navegación para la oferta de la tienda:

1.  **Exposición Clara de Productos:** Presentando productos alimenticios mediante tarjetas visualmente atractivas y cliqueables.
2.  **Facilitar el direccionamiento:** Redirigir a los usuarios al carrito de compras y acceder a mayor información a traves de botones y referencias.
3.  **Generar confianza con el cliente:** Incluir una sección de reseñas y un formulario de contacto para establecer credibilidad y ofrecer múltiples canales de comunicación.

## Estructura y Tecnologías

El sitio está construido con tecnologías web fundamentales, siguiendo prácticas modernas de diseño adaptable:

* **HTML5:** Estructura semántica clara para facilitar la accesibilidad.
* **CSS3:** Estilos completos con **Flexbox** y **Grid** (para la sección de productos), asegurando un diseño responsive.
* **APIs:** Acceso a productos a traves del consumo de una api que contiene una amplia variedad de productos
* **Diseño Responsive:** La implementación de Media Queries asegura una visualización óptima en dispositivos móviles, cambiando la disposición de los productos de 3 a 1 columna.

## Aclaración imoprtante

* Para propósitos de desarrollo y prueba inicial, parte de la información fue tomada como referencia de una página amiga del rubro. 
* Producto realizado a modo de prueba y demostración de aprendizaje a través del curso de Front - End, sin fines comerciales.
